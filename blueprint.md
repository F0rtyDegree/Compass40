# Blueprint: Спорткарта

## Обзор

Это мобильное приложение "Спорткарта", разработанное для помощи в навигации по спортивным картам. Пользователи могут загружать изображения карт, привязывать их к GPS-координатам, отмечать цели и отслеживать свое местоположение и маршрут.

## Реализованные функции

### Версия 3.2.5 (Базовая)

*   **Загрузка карты**: Пользователи могут выбрать изображение карты из галереи своего устройства.
*   **Привязка карты**:
    *   Возможность добавлять "якоря" — точки, связывающие пиксель на изображении с реальными GPS-координатами.
    *   Координаты для якоря можно взять из текущего местоположения GPS или вставить из буфера обмена.
    *   Для калибровки карты требуется как минимум два якоря.
*   **Отображение трека**: Приложение рисует сплошную линию, отображающую пройденный пользователем путь (`userPath`).
*   **Управление картой**: Поддерживаются жесты для панорамирования, масштабирования и вращения карты.
*   **Цели**:
    *   Возможность устанавливать цели на карте.
    *   Отображение линии до активной цели.
    *   Отображение расстояния и азимута до активной цели.
*   **GPS и Компас**:
    *   Отображение текущего местоположения пользователя на карте.
    *   Интеграция с компасом устройства для отображения направления движения.
*   **Сохранение состояния**: Проект (карта, якоря, цели, путь) и последняя трансформация карты сохраняются между сессиями.
